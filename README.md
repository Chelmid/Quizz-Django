# Quizz-Django

Projet quizz

Fonctionnement
* thème
* choix du difficulté
* nombre de question

Les languages et api utilissés :
* python ( Django )
* Tailwind
* API question ( https://the-trivia-api.com/ )

Commande pour démarrer:
> py manage.py runserver
